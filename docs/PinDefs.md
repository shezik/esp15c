# Pin Definitions

![DOIT32 devkit v1 pinout](pinoutDOIT32devkitv1.png)

## Display LX-12864L-1 (SPI)
| Arduino | Description  | Display |
|---------|--------------|---------|
| D18     | SCL          | 4       |
| D19     | A0 (D/C)     | 3       |
| D23     | SDA (MOSI)   | 5       |
| D5      | CS           | 1       |
| EN      | RST          | 2       |
| 3V3     | VDD          | 6       |
| GND     | GND          | 7       |
